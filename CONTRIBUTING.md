# Contributing to Aurora

Thanks for your interest in contributing! Please follow these steps:

- Fork the repository and create a feature branch: `git checkout -b feature/your-feature`
- Run tests and linters (if any) and ensure the app builds:
  - `pnpm --prefix ui install`
  - `pnpm --prefix ui run build`
- Open a Pull Request with a clear description of your changes and link to any related issue.
- For major changes, open an issue first to discuss the design.

Maintainers may request changes; we appreciate prompt updates to keep reviews moving.
